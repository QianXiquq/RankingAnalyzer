# RankingAnalyzer - 成绩排名分析器

## 项目描述

一款用于可视化分析学生历次考试成绩与排名趋势的桌面软件。通过导入CSV数据，自动生成清晰的折线图，帮助快速定位优劣势科目和成绩波动情况。

## 环境要求

- Python 3.8+
- 依赖库：
  - PyQt5
  - pandas
  - matplotlib
  - numpy

## 使用说明

### 1. 数据准备
创建包含以下列的CSV文件：
- `exam` (考试日期，格式：YYYY-MM-DD)
- `subject` (科目名称)
- `score` (分数)
- `rank` (排名)

### 2. 数据导入
- 点击"导入CSV"手动选择文件
- 或创建`data.csv`文件放在程序旁自动加载

### 3. 查看分析
- **科目趋势**：比较各科分数变化
- **总分趋势**：查看总分波动
- **排名趋势**：查看排名变化（数字越小位置越高）

### 4. 记录备注
- 右侧面板可记录考试总结
- 点击"编辑备注"修改内容
- 点击"保存备注"永久存储
